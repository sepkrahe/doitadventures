# Do It! Adventures — Brand Assets

Public hosting for the **SAFARI CREW** sunset badge, used in the daily
trip-digest emails. Files here must stay at the **repository root** so the raw
URLs below keep working.

## ⚠️ Upload instructions
Commit **both PNGs to the root** of `https://github.com/sepkrahe/doitadventures`
on the **`main`** branch (drag-and-drop in the GitHub web UI → "Commit changes").
Do not move them into a subfolder, or the email `<img>` URLs will break.

## Files & their raw URLs
| File | Size | Raw URL (use this in email `<img src>`) |
|---|---|---|
| `logo-badge-email.png` | 260×260, transparent | `https://raw.githubusercontent.com/sepkrahe/doitadventures/main/logo-badge-email.png` |
| `logo-badge.png` | 400×400, transparent (hi-res) | `https://raw.githubusercontent.com/sepkrahe/doitadventures/main/logo-badge.png` |

The three email templates already point at the **`logo-badge-email.png`** raw URL.
As soon as you commit the file, the badge appears in every client (incl. Gmail).

> Tip: after committing, open the raw URL in a browser to confirm it loads. If you
> ever rename the repo, default branch, or file, update the `<img src>` in the
> templates to match.
